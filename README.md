# Aether Theme

An ethereal, versatile web palette featuring Ash Blue as the primary anchor, with comprehensive Light and Dark modes.

## Overview
This repository contains the `theme.json` definition and a showcase website demonstrating the **Aether** theme in action.

## Structure
- `theme.json`: The single source of truth for design tokens.
- `index.html`: Showcase landing page.
- `style.css`: Vanilla CSS implementation using CSS Variables and glassmorphism.
- `script.js`: Dynamic palette generation with "Falling Leaves" layout.

## Usage
Simply open `index.html` in your browser to view the showcase.

## Features
- **Falling Leaves Layout**: Color swatches are scattered organically.
- **Interactive**: Hover to focus, click buttons to cycle colors.
- **Dark Mode**: Fully supported with deep blue/slate tones.
